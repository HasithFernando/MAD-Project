## ğŸ§ª Getting Started

Follow these steps to run Thriftale locally:

1.  **Clone the repository**
    
    ```bash

    ```

    ```bash
    flutter pub get
    ```

    -   Add `google-services.json` (for Android)

4.  **Run the app**
    
    ```bash
    flutter run
    ```
